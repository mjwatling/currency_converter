# public/ — Static Assets

Files served directly by Next.js at the root URL path (`/`). No routing or server-side processing needed.

## Files

| File | Purpose |
|------|---------|
| `favicon.ico` | Browser tab icon for CurrencyXchange |

## Notes

- Add images, fonts, or other static files here to serve them at `/filename.ext`
- `.next/` build artifacts and `node_modules/` are excluded via `.gitignore`
- `.cache/` directory (persistent rate cache) is also gitignored and lives at project root
